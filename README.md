# Student Grade Management System

A complete Python-based Grade management system for managing students, courses, assignments, GPA calculation, and transcript generation using clean Object-Oriented Programming principles.

---

## Problem Statement

Educational institutions require a structured system to manage student academic records efficiently. Managing assignments, calculating weighted grades, maintaining GPA records, and generating official transcripts manually can become error-prone and difficult to scale.

The objective of this project is to build a centralized Student Grade Management System that can:

- Manage multiple students and courses
- Record assignments and exam scores
- Apply category-based weighted grading
- Calculate final course grades
- Compute cumulative GPA
- Generate official academic transcripts

The system should support different assessment categories such as homework, quizzes, midterms, and final exams while maintaining clean, modular, and scalable code architecture.

## Functionalities

- Add and manage students

- Store academic information

### Course Management

### Grade Processing
